  - API 연동 (Axios를 활용한 비동기 처리)  
  - 상태 관리 (Redux Toolkit)

  기술 스택: `Node.js`, `React`, `AWS (EC2, S3, RDS)`  
  주요 기능:  
  - 백엔드 서버 배포 (EC2 사용)  
  - 정적 파일 호스팅 (S3 사용)
